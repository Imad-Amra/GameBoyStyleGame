# Game Boy–Style Island Runner

A 2D arcade game built with **Python (Tkinter + Turtle)**.  
The player collects gold while avoiding an **AI-controlled enemy** that navigates around islands.

## ✨ Features
- **Enemy pursuit logic** with obstacle checks and collision detection  
- **Keyboard and on-screen controls** for movement  
- **Score tracking** with persistent high score (saved in `highscore.txt`)  
- **Reset button** for instant replays  
- **Spawn logic** that avoids forbidden zones for fair play

## ▶️ Run Locally
```bash
python main.py
